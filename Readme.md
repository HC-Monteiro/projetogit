Passo a passo de como utilizar o GIT!!

Teve uma alteração

git init - (dentro da pasta para criar o repositório)
git status - (para entender qual a posicao atual no nosso processo)
git add - (com o nome do nosso arquivo para colocar dentro da area de espera)
git commit -m ( com a mensagem do commit para de fato criar esse ponto de alteracao do arquivo)
